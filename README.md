
This repository contains Python solutions for multiple programming and mathematical problem statements. The implementation focuses on problem-solving, exception handling, algorithm optimization, and logical computation using Python.

Problems Implemented
1. OS Command Executor

A Python program that executes a list of operating system commands and returns the result in a structured dictionary format.

Features
Executes valid OS commands (ls, pwd, df, etc.)
Removes duplicate commands before execution
Handles command execution failures gracefully
Continues execution even if one command fails
Displays output, error message, and execution status


2. Prime Number Family Problem (51)

Implementation of the prime digit replacement problem where part of a number is replaced with the same digit to generate a family of prime numbers.

Concepts Used
Prime number generation
Pattern matching
Mathematical optimization
Algorithmic problem solving


3. Nim Game Problem(301)

Solution for the Nim Game problem using XOR and bitwise operations to determine winning and losing positions.

Concepts Used
Game theory
XOR operations
Bit manipulation
Mathematical reasoning


Technologies Used
Python 3
Subprocess Module
Bitwise Operations
Prime Number Algorithms
